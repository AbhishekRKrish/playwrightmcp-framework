The tests in this project are for an e-commerce website (https://ecommerce-playground.lambdatest.io/).

Key test objectives:
1. Product search and filtering
2. Shopping cart operations
3. User account management
4. Checkout process

Common test patterns:
- Use Page Object Model for better maintainability
- Include assertions for validation
- Handle dynamic loading states
- Use test data effectively

Technical requirements:
- Base URL: https://ecommerce-playground.lambdatest.io/
- Browser: Chromium
- Screenshot on failure
- Video recording for failed tests
- Allure reporting integration
